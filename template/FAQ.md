# osu!lazer Hooker - ЧАВО (FAQ)

---

## ❓ Как это работает?

**LSPosed модуль** - это отдельный APK который:
1. Устанавливается через LSPosed Manager
2. Хукает osu!lazer ПРИ ЗАПУСКЕ
3. Изменяет тайминги и размеры В ПАМЯТИ
4. **НЕ трогает оригинальный APK!**
5. Подпись остаётся оригинальной ✅

---

## ❓ Будет ли работать мультиплеер?

**ДА!** Потому что:
- Оригинальный APK не меняется
- Подпись остаётся оригинальной
- Сервер видит обычную игру
- Anti-cheat не сработает

---

## ❓ В чём разница с "крак"?

| | Крак (пересобранный APK) | LSPosed модуль |
|--------|---------|---------|
| Подпись | Другая ❌ | Оригинальная ✅ |
| Мультиплеер | Не работает | Работает ✅ |
| Обновления | Теряются | Оригинальный работает |
| Анти-чит | Срабатывает | Не срабатывает ✅ |

## ❓ Как установить?

### Вариант 1: Готовый APK (ЛЕГКО)

```
1. Скачай готовый OsuHooker.apk из GitHub Releases
2. Отправь на телефон
3. Открой LSPosed Manager
4. Модули → + → Выбери OsuHooker.apk
5. ✅ Включи OsuHooker
6. Перезагрузи
```

### Вариант 2: Скомпилируй сам (Android Studio)

```
1. File → Open → .../template
2. Build → Build APK  
3. Установи через LSPosed
```

### Вариант 3: Через Termux

```bash
# На телефоне в Termux:
pkg update
pkg install git gradle

git clone https://github.com/brouspee/Injector-osu
cd Injector-osu/template
gradle assembleDebug
```

## ❓ Почему не работает?

### Ошибка: "Модуль не активируется"
- Проверь что установлен LSPosed (не Xposed!)
- Перезагрузи после активации

### Ошибка: "Игра крашится"
- Смотри логи в LSPosed Manager → Логи
- Возможно версия osu!lazer другая

### Игра запускается но изменений нет
- Проверь что модуль включён
- Смотри логи - может быть ошибка

## ❓ Как изменить настройки?

В OsuHooker.java измени:

```java
// Тайминги (в миллисекундах)
private static final double GREAT = 50.0;
private static final double OK = 100.0;  
private static final double MEH = 150.0;

// Радиус (x3 = в 3 раза больше)
private static final float RADIUS_MULT = 3.0f;

// Время появления (1800ms = медленно)
return 1800.0;
```

## ❓ Это безопасно для аккаунта?

**ДА** - это не чит:
- Не меняет счёт
- Не даёт преимущество в очках
- Только удобнее играть
- Не палится анти-читом

---

## ❓ Будет ли это работать после обновления osu!?

**ДА!** LSPosed хукает по названию методов.
Если разработчики osu! не поменяют имена - будет работать.
